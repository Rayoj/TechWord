## Authentication

  </Card>
  <Card
    title="/authentication/token/new"

   a TMDB user login.
  </Card>
  <Card
    title="/authentication/session/new"
    icon="square-3"

  </Card>
  
</CardGroup>

## Account

<CardGroup cols={1}>
  <Card
    title="/account/{account_id}/favorite"
    icon="square-1"
    href="https://rayoco.mintlify.app/openapi/account/mark-as-favorite"
  >
    This endpoint allows you to mark a movie or TV show as a favourite.
  </Card>
  <Card
    title="/account/{account_id}/watchlist"
    icon="square-2"
    href="https://rayoco.mintlify.app/openapi/account/add-to-watchlist"
  >
   This endpoint allows you to add a movie or TV show to your watchlist.
  </Card>
  <Card
    title="/account/{account_id}"
    icon="square-3"
    href="https://rayoco.mintlify.app/openapi/account/lists"
  >
  This endpoint allows you to get a users list of custom lists.
  </Card>
  
</CardGroup>

## Certifications
<CardGroup cols={1}>
  <Card
    title="/certification/movie/list"
    icon="square-1"
    href="https://rayoco.mintlify.app/openapi/collections/details"
  >
    This endpoint allows you to get an up-to-date list of the officially supported movie certifications on TMDB.
  </Card>
   <Card
    title="/certification/tv/list"
    icon="square-2"
    href="https://rayoco.mintlify.app/openapi/collections/images"
  >
   This endpoint allows you to get an up-to-date list of the officially supported TV show certifications on TMDB.
  </Card>
  </CardGroup>
