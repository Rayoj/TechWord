Welcome to the official TMDb API documentation! Access a vast, community-driven movie and TV database. Whether you're a developer, film enthusiast, or business, our powerful API helps you integrate rich entertainment data with ease.


- You will need to pick a language.
- Get your credentials such as your API Access Key and API Key for authentication and authorization.
- Please note that the API registration process is not optimized for mobile devices so you should access these pages on a desktop computer and browser.


## API Base URL

All endpoints in this documentation use this base URL:  
``https://api.themoviedb.org/3``

## Content-Type

Endpoints expect and return data in the `json/application` format.


## Getting Started

To get started with the TMDB API, you need to follow the following steps:


### Step 1  

┃  
┃ **[Create an account](https://www.themoviedb.org/signup) to get started**.   
┃ 

### Step 2  

┃  
┃ **After creating an account, [get your API Key](https://www.themoviedb.org/settings/api).**    
┃ 


### Step 3 

┃  
┃ **Know the [configuration methods](https://developer.themoviedb.org/reference/configuration-details).** **They retrieve static data lists for the database.**
┃ 


### Step 4  

┃  
┃ **Understand [Authentication Basics](https://developer.themoviedb.org/docs/authentication-application).**  
┃
